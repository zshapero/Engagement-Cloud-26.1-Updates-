What Is New in the 26.1 Release

The following updates are included in the 26.1 release of Engagement Cloud.

  • Intuitive user interface — Accurate, concise, and actionable insights allow marketers to better understand and respond to trends in subscriber experience and service adoption. Simplified segment and campaign management helps marketers more efficiently gather insights, create segments, and deploy campaigns.
  • Data quality and file upload dashboards — New Configuration page allows marketers to view billing data metrics, download endpoint and demographic matching results, and upload ZIP code filter and subscriber lists directly within Engagement Cloud.
  • Business performance dashboards — New Business Insights dashboard provides visibility into the health of the organization, allowing marketers to take immediate, proactive action where needed.
  • Segment builder — Create targeted segments with a robust set of filters organized by data category, including a map filter for geographic segmentation. Pre-built demographic and subscriber segments for key personas such as Gamers, Work from Home professionals, and Unhappy Subscribers are also included.
  • Simplified campaign management — Streamlined, single-page campaign builder allows marketers to quickly create and deploy campaigns, refine audiences by including or excluding segments, and select flexible performance metrics to track campaign results.
